Java Enterprise Edition

Java EE is modern and rich ecosystem suitable for building applications of any size and focus. Learning, coding and maintaining Java EE applications is exceptionally easy.

Getting started

Easy

Java EE is easy to learn and makes everyday coding a piece of cake. Type-safe dependency injection, no configuration hell, thin and lightweight applications ... the list goes on. Java EE is rock solid in its ultimate goal to let the developer focus on busines logic.

Code long & in prosper

Backwards compatibility is a matter of course. There are many open-source mutually compatible solutions that just work. When a new version comes out, there is no need to adjust your application. Future of your application is safe with Java EE.

Cloud ready

Java EE is ready to be deployed to the cloud and offers far more than traditional "boot" style. In a world where self-contained microservices are the actual trend, Java EE truly shines. Fast development, quick redeployments without introducing complex tools ... and yes, there are Docker images for everything.
